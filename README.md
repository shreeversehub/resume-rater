# 📖 Overview

Resume Rater is a **Python-based rule-driven resume analysis system** that compares a candidate's resume with a given job description. It extracts candidate information, calculates a resume match score, identifies possible fake resume indicators, recommends suitable company categories, and estimates an expected salary package.

This project demonstrates the practical application of **Python**, **Regular Expressions**, and **Rule-Based Decision Making** for automating resume evaluation.
